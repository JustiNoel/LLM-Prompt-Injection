Architecture

4 Layers of Defense

Every request passes through all four layers before reaching your LLM — and before the response reaches your user.

LAYER 01
🔍
Input Classification
Pattern matching + heuristic scoring detects injection attempts, jailbreak templates, and role-override attacks before anything else runs.
LAYER 02
🧹
Context Sanitization
Strips prompt delimiters, neutralizes override phrases, removes encoded payloads, and wraps input in safe boundaries.
LAYER 03
🔐
Prompt Integrity
HMAC-signs and verifies your system prompt hasn't been tampered with. Catches structural violations before the LLM call.
LAYER 04
👁
Output Monitoring
Scans LLM responses for jailbreak success signals, system prompt leakage, PII exposure, and harmful content patterns.
