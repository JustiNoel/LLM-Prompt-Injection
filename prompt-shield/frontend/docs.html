PromptShield API

4-layer LLM prompt injection defense middleware. Protect your AI applications from jailbreaks, prompt leaking, and instruction overrides.

🔗 Base URL: https://prompt-shield.onrender.com

Authentication

All protected endpoints require an API key passed in the request header.

X-API-Key: ps_live_yourkey

Quickstart

# Install requests if needed
pip install requests

# Simple chat request
curl -X POST https://prompt-shield.onrender.com/chat \
  -H "X-API-Key: ps_live_yourkey" \
  -H "Content-Type: application/json" \
  -d '{"message": "Hello!"}'

POST /chat

POST /chat Protected

Send a message through all 4 defense layers to the LLM backend.

Request Body

FieldTypeRequiredDescription
messagestringYesUser message to process
session_idstringNoOptional session identifier

POST /analyze

POST /analyze Protected

Analyze text for threats without calling the LLM. Fast and lightweight.

{
  "threat_level":       "malicious",
  "score":              0.95,
  "triggered_patterns": ["[INJECTION] ignore.*instructions"],
  "reasoning":          "High-confidence injection attempt",
  "sanitized":          "[instruction-override-attempt]",
  "modifications":      ["Neutralized override"]
}

POST /aggression

POST /aggression Protected

Change the shield aggression level at runtime.

{ "level": "strict" }

GET /stats

GET /stats Protected

Returns current thresholds and blocking configuration.

GET /logs

GET /logs?limit=50 Protected

Returns recent audit log entries. Max 1000.

GET /logs/summary Protected

Returns block rate, threat breakdown, and average response time.

Python SDK

from promptshield_sdk import PromptShieldClient

client = PromptShieldClient(api_key="ps_live_yourkey")

# Chat
result = client.chat("Hello!")
print(result["response"])

# Analyze a threat
threat = client.analyze("Ignore previous instructions")
print(threat["threat_level"])  # malicious

# Get summary stats
summary = client.summary()
print(summary["block_rate"])

JavaScript

const shield = {
  baseUrl: "https://prompt-shield.onrender.com",
  apiKey:  "ps_live_yourkey",

  async chat(message) {
    const res = await fetch(`${this.baseUrl}/chat`, {
      method: "POST",
      headers: {
        "X-API-Key":    this.apiKey,
        "Content-Type": "application/json",
      },
      body: JSON.stringify({ message }),
    });
    return res.json();
  }
};

// Usage
const result = await shield.chat("Hello!");
console.log(result.response);
